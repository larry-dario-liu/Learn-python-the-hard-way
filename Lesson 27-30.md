### Lesson 27-30  
#### 逻辑运算  
  if条件语句，语法如下:  
  ```
  if 判断语句：
       判断成立时运算
  if my_age >= 20:
      print "you should work hard"
  ```  
  判断语句后一定要有冒号、要有冒号、要有冒号，条件运算语句要后缩进。  
  if条件语句的变式：  
  ```
  if 判断语句a:  
      判断语句a成立时运算
  elif 判断语句b:  
      判断语句b成立时运算  
  else:  
      a、b以外，其他情况下运算
  ```

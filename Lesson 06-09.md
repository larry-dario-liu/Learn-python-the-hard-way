### Lesson 6-9
#### string and text
这几节其实是文本与变量的深化练习，格式符、变量、文本之间可以做各种嵌套。
- 文本之间用加号"+",可以之间连接输出，类似VBA的"&"的功能。  
```
print "larry is " + "gorgeous"
```
- 文本加乘号*接数字表示重复
```
print "dida" * 3
```

- 文本格式符```\n```表示文本换行
- 番外   
即使输入了```# -*- coding: utf-8 -*- ```,最开始输入中文会显示繁体的乱码，把notepad++的encoding选项更改一下就好。  
注意：作为一种数据格式，逻辑符号True和False都首字母都要大写；要打印中文只能用%s不能用%r。

3. Lesson 1

本节作者就是演示了基本的操作过程：
- 在编辑器新建Python文件，输入代码,对就是那个最经典的代码：
```
print "Hello world!"
```
- 在终端输入命令，运行Python文件，呈现打印结果，命令:
```
python ex1.py
```
如果是在vba里，达到同样效果也很简单，不多比Python多2条代码：
```
sub
msgbox "Hello world"
end sub
```
文本都用双引号框起来了。

- 注释
study drill提示去练习“#” 前置代码的效果，这其实就是注释，VBA则是带引号“'”前置.

## Learn-python-the-hard-way
### Lesson 12
##### 输入提示  
本节很简单，介绍了函数raw_input()的输入提示
简单讲，```a=raw_input("输自己的名字")```等价于：  
```
print "输自己的名字"
a=raw_input()
```
####课后思考  
在Python终端，这里是powershell输入```python -m pydoc```,会显示python帮忙文档工具，只不过是通过代码实现，例如：  
需要查找class在Python的含义，输入：  
```
python -m pydoc class
```
也就是说多了一条搜索学习的方法。
